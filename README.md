# 🧑‍🤝‍🧑 Face Finder - Back End 🧑‍🤝‍🧑

This is the backend for the Face Finder application. It provides secure user registration and login features, interacts with the Clarifai API for face detection, and stores user information in a Postgres database.

## 🛠️ Built With 🛠️

-   [Node.js](https://nodejs.org/)
-   [Express](https://expressjs.com/)
-   [Knex.js](http://knexjs.org/)
-   [Postgres](https://www.postgresql.org/)
-   [Clarifai API](https://www.clarifai.com/)
-   [bcrypt](https://www.npmjs.com/package/bcrypt)
